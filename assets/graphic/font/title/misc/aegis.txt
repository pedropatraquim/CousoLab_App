Aegis Truetype Font for Windows

2010 Iconian Fonts - Daniel Zadorozny

http://www.iconian.com/

This font comes with the following 9 versions: Regular, Italic, Condensed, Condensed Italic, Expanded, Expanded Italic, 3D, 3D Italic and Leftalic.

This font has several dingbats in the extended character set - use of a character map program will help you explore and use this font.
